2. Privacy Policy

Protecting your server data is our highest priority. The App only processes and stores data that is absolutely essential to deliver its features:

  • Stored Data: We store unique Discord IDs (Server IDs, Channel IDs, Role IDs) and generated premium keys in a secured database to load your custom server configurations.
  • Server Logging Feature: The App includes an advanced logging system (e.g., deleted/edited messages, member joins/leaves, moderation actions). This feature is completely optional and turned off by default. It only activates if a server administrator explicitly configures a log channel using the /setlogchannel command.
  • Data Processing: Message content is processed strictly in real-time to trigger configured features (like logging or auto-reactions). We never permanently store private chat histories on our servers or sell any data to third parties.
  • Data Deletion: If the App leaves a Discord server, all stored configurations and logging settings associated with that server can be completely removed upon request.
